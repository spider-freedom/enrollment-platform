# 招生宣传报名平台 — 部署说明

## 一、队友电脑需要的环境

| 软件 | 版本 | 下载 |
|------|------|------|
| JDK | 17+ | https://adoptium.net |
| Maven | 3.8+ | https://maven.apache.org |
| Node.js | 18+ | https://nodejs.org |
| MySQL | 8.0 | https://dev.mysql.com |

装完后验证：
```bash
java -version    # 显示 17.x
mvn -version     # 显示 3.8+
node -v          # 显示 v18+
npm -v           # 显示 9+
mysql --version  # 显示 8.0.x
```

## 二、打包项目发给他

### 方式 1：复制文件夹（最简单）

把你的 `E:\enrollment-platform` 整个文件夹拷到 U 盘/网盘发给他。

### 方式 2：用 Git（推荐）

```bash
# 你的电脑
cd E:\enrollment-platform
git init
git add .
git commit -m "init"
# 上传到 GitHub/Gitee，队友 clone
```

## 三、队友拿到后要做的

### 第 1 步：安装依赖

```bash
# 后端依赖（自动下载）
cd enrollment-backend
mvn clean compile

# 前端依赖
cd ../enrollment-frontend
npm install
```

### 第 2 步：创建数据库

打开 MySQL，执行 `database/init.sql`：
```bash
mysql -u root -p < database/init.sql
```
或者用 Navicat/Workbench 打开 `init.sql` 运行。

> ⚠️ 如果他的 MySQL root 密码不是你设置的 `123456`，需要改两个地方：
> 1. `enrollment-backend/src/main/resources/application.yml` 里的 `password: 123456` 
> 2. `database/init.sql` 里的连接信息不重要，他用工具执行就行

### 第 3 步：改密码哈希

`init.sql` 里的密码哈希值（`$2a$10$N...`）在你的电脑上能用，但如果队友用不同版本的 BCrypt 可能不匹配。最保险的方法：

启动后端后，用注册功能注册一个新账号，或者：
```bash
# 用 backend 自带的 BCrypt 生成正确哈希
# 或者直接让队友在数据库里执行：
UPDATE sys_user SET password='$2b$12$fbKC6xEPicrekDy8wB5a0.koj95WTlXvFLg59sdafHQmNytLpdWpO';
```

### 第 4 步：启动后端

```bash
cd enrollment-backend
mvn spring-boot:run
```

看到 `Started EnrollmentApplication` 就是启动成功了。后端跑在 `http://localhost:8080`。

### 第 5 步：启动前端

```bash
cd enrollment-frontend
npm run dev
# 或
npx vite --port 3000
```

打开 `http://localhost:3000`，用 `school1` / `123456` 登录。

## 四、常见问题

| 问题 | 解决 |
|------|------|
| 后端启动报数据库连接失败 | 检查 MySQL 是否启动，密码是否正确 |
| 前端页面空白 | 检查后端是否启动，`npm install` 是否执行过 |
| 登录报 500 | init.sql 有没有成功执行 |
| 端口被占用 | 改 `application.yml` 的 `port: 8080` 为其他端口 |
| 中文乱码 | 建数据库时加 `DEFAULT CHARSET utf8mb4` |

## 五、一键启动脚本

队友拿到项目后，双击运行：

**start.bat**（放在项目根目录）
```bat
@echo off
echo 启动后端...
start "Backend" cmd /c "cd enrollment-backend && mvn spring-boot:run"
echo 等待后端启动...
timeout /t 25 /nobreak >nul
echo 启动前端...
start "Frontend" cmd /c "cd enrollment-frontend && npm run dev"
timeout /t 5 /nobreak >nul
start http://localhost:3000
echo 完成！
pause
```
